Git config


$ git config --global user.name "John Doe"
$ git config --global user.email johndoe@example.com
$ git config --list

Bien! Ya no serás anonimous ;-)

Primeros commits

Iniciar un repo


						$ git init

Solo hace falta para iniciar un repo

Clonar un repo


						$ git clone git@git.assembla.com:repositorio.git

Un poco de teoría

Mi primer commit


$ touch README.md
$ git add README.md
$ git commit -m "Hola mundo este es mi primer commit!"

Segundo commit

Hacemos cambios en el README.md


$ git status
$ git commit -a

Ya tenemos 2 commits

Enviar al remote:origin


$ git pull origin/master
$ git push origin/master

Retrocedamos en el tiempo


$ git checkout 2ed6dd132718a42b524f14ae216df0d78d2a1144

Back to the future


$ git checkout master

Branches, haz tus propias ramas.

Creamos nuestra rama


$ git checkout -b rama_nueva

Igual que "master, pero menos peligrosa


$ git commit -am "primer commit de esta rama"...

La mandamos a origin


$ git push -u origin rama_nueva

Lléveme a master, por favor


$ git checkout master

Merge = juntar ramas


$ git merge "rama_solo_para_nosotros"

GIT stash

Estos cambios no los quiero commitear, pero...


$ git stash
$ git stash save "blah blah blah"
$ git stash list
$ git stash pop stash@{1}
